# IF677-Projects

Projects made for Software Infrastructure subject (IF677) from Computer Engineering degree on CIn-UFPE which goal is to apply and exercise the passed concepts and knowledge about Assembly language, booting process and parallel computing using [pthread library](https://www.cs.cmu.edu/afs/cs/academic/class/15492-f07/www/pthreads.html) from C/C++ language. The projects consists on an Assembly problems list, an Assembly boot application and a pthread problems list.
